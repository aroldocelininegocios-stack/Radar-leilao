# 🎯 Radar de Leilões Municipais — Guia de Instalação

Siga este guia do zero. Não precisa saber programar.
Tempo estimado: 30 a 45 minutos.

---

## O que você vai precisar (tudo gratuito)

- Conta no GitHub: https://github.com
- Conta no Supabase: https://supabase.com
- Conta na Vercel: https://vercel.com
- Chave da API Anthropic: https://console.anthropic.com

---

## PASSO 1 — Criar conta e projeto no GitHub

1. Acesse https://github.com e crie uma conta (se não tiver)
2. Clique em "New repository" (botão verde)
3. Nome: `radar-leiloes`
4. Deixe como "Public"
5. Clique em "Create repository"
6. Faça upload de todos os arquivos desta pasta para o repositório
   (clique em "uploading an existing file" na página do repositório)

---

## PASSO 2 — Criar banco de dados no Supabase

1. Acesse https://supabase.com e crie uma conta
2. Clique em "New Project"
3. Dê um nome: `radar-leiloes`
4. Escolha uma senha forte (anote!)
5. Região: South America (São Paulo)
6. Clique em "Create new project" e aguarde ~2 minutos

### Criar a tabela de leilões:

1. No menu esquerdo, clique em "SQL Editor"
2. Cole e execute o seguinte SQL:

```sql
create table leiloes (
  id bigint generated by default as identity primary key,
  prefeitura text,
  tipo text,
  distancia_sp integer,
  data_leilao text,
  hora_leilao text,
  link text,
  leiloeiro text,
  populacao bigint,
  valor_estimado numeric,
  descricao text,
  criado_em timestamptz default now()
);
```

3. Clique em "Run"

### Pegar suas chaves do Supabase:

1. Vá em "Settings" → "API"
2. Copie:
   - **Project URL** → é o `NEXT_PUBLIC_SUPABASE_URL`
   - **anon public** → é o `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - **service_role secret** → é o `SUPABASE_SERVICE_ROLE_KEY`

---

## PASSO 3 — Pegar chave da Anthropic (IA)

1. Acesse https://console.anthropic.com
2. Faça login ou crie conta
3. Vá em "API Keys" → "Create Key"
4. Copie a chave — começa com `sk-ant-...`
   Essa é o `ANTHROPIC_API_KEY`

⚠️ O plano gratuito da Anthropic dá créditos iniciais.
   Uso estimado do site: menos de $5/mês.

---

## PASSO 4 — Publicar na Vercel

1. Acesse https://vercel.com e crie conta (use "Continue with GitHub")
2. Clique em "Add New Project"
3. Selecione o repositório `radar-leiloes`
4. Antes de clicar em Deploy, vá em **Environment Variables** e adicione:

| Nome | Valor |
|------|-------|
| NEXT_PUBLIC_SUPABASE_URL | (sua URL do Supabase) |
| NEXT_PUBLIC_SUPABASE_ANON_KEY | (sua chave anon) |
| SUPABASE_SERVICE_ROLE_KEY | (sua service role key) |
| ANTHROPIC_API_KEY | (sua chave Anthropic) |
| CRON_SECRET | (invente uma senha, ex: MinhaS3nha2024) |

5. Clique em "Deploy"
6. Aguarde ~2 minutos. Seu site estará no ar!

---

## PASSO 5 — Fazer a primeira busca de leilões

O site busca leilões automaticamente toda segunda-feira às 7h.
Para buscar agora pela primeira vez:

1. Abra seu site (ex: https://radar-leiloes.vercel.app)
2. Na barra de endereço do navegador, acesse:
   ```
   https://SEU-SITE.vercel.app/api/cron-buscar-leiloes?secret=MinhaS3nha2024
   ```
   (use a mesma senha que você colocou em CRON_SECRET)
3. Deve aparecer: `{"sucesso":true,"total":X,...}`
4. Volte para o site e os leilões estarão lá!

---

## Como funciona a atualização automática

Todo site publicado na Vercel recebe um cron job configurado em `vercel.json`.
O arquivo já está configurado para rodar toda segunda-feira às 7h da manhã.

A IA (Claude) vai:
1. Pesquisar nos Diários Oficiais municipais
2. Identificar leilões de veículos e inservíveis da semana
3. Calcular distância, valor estimado e score
4. Salvar automaticamente no banco de dados Supabase
5. Exibir no site atualizado
